Installation Instructions:
To make this system run, you must have an Wamp (http://www.wampserver.com/en/) server and MySQL (https://www.mysql.com/) database installed on your running environment. Please follow the links provided to download the free version of the software. After Wamp and MySQL are installed, please parse the .SQL files provided with the system to establish the database via Wamp server.
